# esp-matter-devices

A collection of Matter devices i've built using [esp-matter](https://github.com/espressif/esp-matter/) framework for esp32 family SoCs

Important: these are quick cheap and dirty 'working enough' implementations, so consider them more like a source of useful insights, than ready to use polished products

## [lunalamp-matter](/lunalamp-matter/)

Extended Color Light device for RGBWW strips, port of my HomeKit [lunalamp-esp32](https://github.com/thisiseth/lunalamp-esp32/)
